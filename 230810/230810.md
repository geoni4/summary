## spring

- Controller
  - @Controller / @RestController
  - @RequestMapping
  - view 결정
  - controller가 하는 내용을 method name으로 작성

- Service / ServiceImpl
  - @Service
  - service 내용을 method name으로 작성
  - transaction 단위에 @Transactional 어노테이션을 달아주면 DB에 원자성이 적용된다(rollback or commit)


- DAO / MyBatis
  - mapper
  - @Mapper / @Repository
  - query에 가깝게 method name 작성
